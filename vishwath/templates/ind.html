UF Scheduling System

Welcome to the UF Scheduling System!



This project was created to provide a scheduling system for UF students.
The project provides the following features:

  • UF bus routes to UF map comparison
  • Database storage of schedules, including add, edit, and delete capability
  • Ability to add preexisting UF events to schedule
  • Generation of a potential bus schedule based on stored schedules
  • Google Maps Integration, with live bus tracking
